## 2) Що зроблено понад план

1. Автоматична валідація інтеграцій при відкритті вкладки (без ручної кнопки “Перевірити підключення”).  
2. Автооновлення monitor status/config у Settings.  
3. Додатковий захист від дублювання в UI (dedupe по `source_id:tg_message_id`) та нормалізація source URL.  
4. Додаткові guard-механізми для AI queue (claim `pending -> processing`, error-status без циклічних повторів).

## 3) Що ще не виконано / потребує підтвердження

1. **E2E навантажувальна верифікація на реальному Pi** (Telethon + Anthropic API) у продовж 24h без ручного втручання.  
2. **Автотести** (unit/integration) для AI worker, queue-claim конкуренції, і правил інгесту по часовому вікну.  
3. **Метрики/алерти витрат AI** (cost dashboard, budget caps, rate limiting policy).  
4. **Розширений retry policy** для тимчасових API-помилок Anthropic (зараз обрано conservative-поведінку: fail-fast у `error` для уникнення зайвих викликів).
